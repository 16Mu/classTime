## 快速开始

### 首次使用
1. **下载安装**
   - 从 GitHub Releases 下载最新 APK
   - 允许"未知来源"应用安装
   - 授予必要权限（通知、存储等）

2. **启动应用**
   - 打开应用进入引导流程
   - 选择你的学校
   - 通过 WebView 登录教务系统
   - 等待课表导入完成

3. **基本设置**
   - 设置课前提醒时间
   - 配置自动更新频率
   - 添加桌面小组件

## 功能介绍

### 智能课表导入
- **支持学校**：27 所高校教务系统
- **登录方式**：WebView 模拟登录，Cookie 加密存储
- **导入格式**：自动识别 JSON/HTML 格式
- **更新机制**：定时自动更新 + 手动更新

### 课程管理
- **周视图**：清晰展示每周课程安排
- **课程编辑**：支持增删改查操作
- **颜色标记**：10 种预设颜色区分课程
- **多学期**：支持多个学期课表管理

### 智能提醒
- **课前提醒**：可自定义提醒时间（默认15分钟）
- **下节课倒计时**：实时显示下节课时间
- **状态感知**：智能识别上课中、课间、放学等状态
- **桌面小组件**：无需打开应用即可查看课程

### 独家功能
- **课程冲突检测**：自动识别时间冲突，提供调课建议
- **多教室管理**：同一课程不同时间可设置不同教室
- **快速导入**：独创快速导入功能，大幅减少导入时间
- **临时调课**：灵活的课程时间调整功能

## 常见问题

### 导入问题
**Q：为什么找不到我的学校？**
A：目前支持27所学校，如果不支持你的学校，可以在 GitHub Issues 中申请适配。

**Q：登录失败怎么办？**
A：检查教务系统账号密码是否正确，确保网络连接正常，部分学校需要验证码。

**Q：课表导入不完整？**
A：可能是教务系统数据格式变化，可以尝试手动导入或反馈问题。

### 使用问题
**Q：提醒不响怎么办？**
A：检查应用通知权限是否开启，确保系统没有限制应用后台运行。

**Q：小组件不更新？**
A：重启设备或重新添加小组件，检查应用是否有省电优化限制。

**Q：课程冲突怎么处理？**
A：应用会自动检测冲突并提供调课建议，也可以手动调整课程时间。

### 技术问题
**Q：应用耗电快？**
A：自动更新功能会消耗一定电量，可以降低更新频率或关闭自动更新。

**Q：数据会丢失吗？**
A：所有数据本地存储，卸载应用前建议导出备份。
