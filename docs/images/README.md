### 图片规格要求
- **尺寸**：建议 1080×1920 像素或按比例缩放
- **格式**：PNG 格式，支持透明背景
- **大小**：每个文件不超过 2MB
- **命名**：使用英文小写，用下划线分隔

### 截图建议
- 使用真机截图，避免模拟器边框
- 确保界面内容清晰可见
- 包含状态栏和导航栏
- 避免显示敏感个人信息
- 使用深色和浅色主题各一张
